This update adds cross-tenant multiplayer support for dedicated servers, making it easier to connect and collaborate across organizations. It also brings key features from the latest Bedrock updates, including craftable nametags, new items like the Golden Dandelion, expanded baby mob variants, and spawn eggs that hatch babies. Players can explore new environments such as Sulfur Caves and Springs, experiment with new block sets like Sulfur and Cinnabar, and benefit from added accessibility features like closed captioning—alongside ongoing bug fixes and performance improvements. \
 

Note: The versioning for Minecraft Education has changed to a new format that starts with the year followed by the release number. 

 

## Cross Tenant support for Dedicated Server (Coming soon)

If your IT Admin enables the capability, schools can now invite other schools to join their worlds! We'll update this page with more info soon along with some steps, so keep an eye out or hit Follow on this page!

 

## Gameplay Updates

This update has the Minecraft Bedrock changes from v1.21.131 to v26.30, including:

### Tiny Takeover

- New item: Golden Dandelion
- New versions of most baby mobs
- Spawn eggs hatch babies 

### Chaos Cubed

- Closed Captioning
- New block sets: Sulfur, Cinnabar
- New locations: Sulfur Caves, Sulfur Springs
- Geysers and Sulfur Spikes too
