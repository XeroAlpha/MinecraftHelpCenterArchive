To view the minimum and recommended requirements for Minecraft: Java Edition, go to the [Minecraft: Java & Bedrock Edition for PC Deluxe Edition](https://www.minecraft.net/en-us/store/minecraft-deluxe-collection-pc?tabs=%7B%22details%22%3A1%7D#accordionv1-b6c8df09da-item-7739893325) product listing and scroll to the **Java System Requirements** section.

For optimal gameplay performance and access to all features and gameplay modes, update your PC or Mac to the most recent OS version or use a device with the most recent OS version. Devices that only meet the minimum requirements may not have access to some features and gameplay modes or run as smoothly as newer devices.

- PC: [Install Windows Updates - Microsoft Support](https://support.microsoft.com/en-us/windows/install-windows-updates-3c5ae7fc-9fb6-9af1-1984-b5e0412c556a)
- Mac: [Update macOS on Mac - Apple Support](https://support.apple.com/en-us/108382)
