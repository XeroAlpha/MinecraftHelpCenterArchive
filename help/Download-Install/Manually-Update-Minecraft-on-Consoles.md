Minecraft will try to automatically update to the latest version if you are connected to the internet. If Minecraft’s version number is lower/older than the one listed in its description on your console store, follow the instructions for your console to manually update.

## How to manually update Minecraft on Xbox consoles

1.  Open **My games & apps.**
2.  Open **Manage**.
3.  Select **Updates**. All games in your library that need an update will be displayed.
4.  Select **Minecraft** or **Update All**.

## How to manually update Minecraft on Nintendo and PlayStation consoles

- **Nintendo Switch:** [How to Update Software on Nintendo Switch](https://en-americas-support.nintendo.com/app/answers/detail/a_id/22767/~/how-to-update-software-on-nintendo-switch)
- **PlayStation 4, 5:** [How to check for updates on PS5 consoles and PS4 consoles](https://www.playstation.com/en-us/support/hardware/playstation-system-software-application-version/)
