Your Friends List in Minecraft: Java Edition helps you coordinate play session with your friends. If you're not familiar yet, [Learn more about the Friends List in Minecraft: Java Edition](./Learn-more-about-the-Minecraft-Java-Edition-Friends-List.md). Here’s how to add friends, cancel requests, remove friends, and decline requests.

## Add a friend

To add your friend, you’ll need their Java profile name. See [View or Change Your In-Game Profile Name in Minecraft](../Manage-Your-Minecraft-Profile/View-or-Change-Your-In-Game-Profile-Name-in-Minecraft.md) for steps to find it.

1.  Open Minecraft: Java Edition.
2.  Select the **Friends** icon from the title screen or pause menu to open the Friends List.
3.  Select **Turn On** if this is your first time using the Friends List.
4.  Enter your friend’s Java profile name in the search field and select **send Friend Request**.

After your friend accepts the request, they’ll show up on your Friends List.

## Cancel a friend request

If you send a request to the wrong player, here’s how to cancel it.

1.  Select the **Friends** icon from the title screen or pause menu to open the Friends List.
2.  Select the **Requests** tab.
3.  Select **cancel request**.

## Remove a friend

If you remove a friend from your Minecraft: Java Edition Friends List, it will also remove them from your Xbox Friends.

1.  Open Minecraft: Java Edition.
2.  Select the **Friends** icon from the title screen or pause menu to open the Friends List.
3.  Find the friend you want to remove and select **Remove Friend**.
4.  Select **Remove** to confirm the change.

## Decline a friend request

1.  Open Minecraft: Java Edition.
2.  Select the **Friends** icon from the title screen or pause menu to open the Friends List.
3.  Select the **Requests** tab.
4.  Find the friend request you do not want to accept and select **Decline**.
