There are a variety of accessibility options to customize the appearance and behavior of chat text for players of all abilities in Minecraft: Java Edition. Microsoft account level chat permissions are managed by [Xbox Settings for Privacy & online safety](https://www.xbox.com/en-US/user/settings/privacy-and-safety?rtc=1&activetab=main:privacytab). See these articles for managing chat permissions:

- [Managing Child Account Social Settings Using Xbox Settings Online](../Account-Settings/Managing-Child-Account-Social-Settings-Using-Xbox-Settings-Online.md)
- [Understanding Minecraft Social Features for Child Safety Online](../General-Safety/Understanding-Minecraft-Social-Features-for-Child-Safety-Online.md)

## Access Chat Settings for Java Edition

1.  Open Minecraft: Java Edition.
2.  Select **Options.**
3.  Select **Chat Settings**.

## Chat Setting Options for Java Edition

- **Chat mode **includes three control options:
  - **Shown: **Display all chat messages (player chat, system messages, and commands).
  - **Commands Only:** Only show command messages.
  - **Hidden:** Hides all chat messages.
- **Colors:** Turns colored text in chat on or off.
- **Weblinks:** Controls whether selectable links in chat are active.
- **Chat text opacity:** Adjusts how transparent the text chat appears.
- **Text background opacity:** Controls transparency of the background behind the chat.
- **Text size:** Changes the size of chat text.
- **Line Spacing:** Adjusts the space between chat lines.
- **Chat Delay: **Controls how long chat messages stay visible.
- **Width: **Controls how wide the chat box appears.
- **Focused Height:** Sets the height of the chat box when the chat window is open.
- **Unfocused Height:** Sets the height of the chat box when the chat window is closed.
- **Narrator:** Turns the in-game narrator on or off for chat messages.
- **Command suggestions:** Controls whether command auto-complete suggestions appear when typing commands.
