*To learn about options available in Java Edition, see* [*Accessibility Settings for Minecraft: Java Edition*](./Accessibility-Settings-for-Minecraft-Java-Edition.md)*.*

We believe everyone should be empowered and enabled to play Minecraft: Bedrock Edition. This article will focus on options available in the Accessibility Menu, for easy access to key settings.  

To open the Accessibility Menu:

- Select **Settings** from the main menu.
- The **Accessibility** options will be displayed by default and is located at the top of the settings list.

## Accessible Menu Navigation and Narration

Bedrock Edition supports navigating menus using a keyboard, controller, or touch controls.

- Arrow Keys: Navigate through menu options.
- Esc Key: Go back to the previous screen or close the current menu.
- Tab Switching: Use the left and right brackets, **\[  \]**, on a keyboard, or **LB/RB** on a controller to switch between tabs.

## Accessible Chat Using Text-to-Speech  

Minecraft: Bedrock Edition includes Text-to-speech options to make in-game chat more accessible. 

- **Text To Speech with Device Settings:** Use your device's built-in text-to-speech settings. When enabled, Minecraft follows the accessibility preferences configured in your operating system. This option is only available on supported devices.
- **Text To Speech For UI:** Toggle **ON** to hear menu options, interface elements, and other UI text spoken aloud.
- **Text To Speech For Chat:** Toggle **ON** to have chat messages read aloud using text-to-speech when they appear in-game.
- **Text To Speech:** Use the slider to adjust the volume of the narration voice.  
- **Chat Instructions on Join:** Toggle **ON** to display a reminder message showing how to open chat whenever you join a world.

## Closed Caption Settings

Closed captions provide visual descriptions of sounds occurring in the game world.  Toggle the **Closed Captions** button to **ON** to unlock the following options:

- **Hide Your Sounds:** Toggle **ON** to hide captions produced by your own actions, such as:
  - Footsteps
  - Eating and drinking
  - Other player-generated sounds
  - Note: Captions for sounds produced by other players and the environment will still appear.  
- **Hide Weather and Ambient Sounds:** Toggle **ON** to hide captions for environmental sounds such as:
  - Rain, thunder, and wind wind
  - Ambient cave sounds
  - Environmental audio effects
- **Location of Closed Captions:**  Choose where closed captions appear on-screen:
  - Top Right
  - Bottom Right
- **Closed Caption Duration:** Adjusts how long captions remain visible after a sound occur

## Accessible Visuals Settings

These options let you choose camera behavior and visual effects that may affect comfort and visibility during gameplay.

- **Camera Shake:** Toggle **ON** to set up camera shake using macros or chat.
- **Hide Sky Flashes:** Toggle **ON** to prevent flashing visual effects caused by lightning strikes or End.
- **Block Dithering:** Toggle **ON** to fade certain blocks when they obstruct the camera, making it easier to maintain visibility.
- **Player and Mob Dithering:** Toggle **ON** to fade nearby entities, such as players and mobs, that obstruct the camera view.
- **Darkness Effect Strength:** Use the slider to adjust how dark the screen becomes when affected by hostile mobs and environmental factors. Lower values reduce the intensity of the darkness overlay.
- **Glint Strength:** Use the slider to control the intensity of the enchantment glint effect. Lower values make the effect more subtle.
- **Glint Speed:** Use the slider to change how quickly the glint animation moves across enchanted items. Lower values result in a slower animation, while higher values increase movement speed.

## Accessible User Interface Settings

The following game UI settings allow you to change the appearance or behavior of game elements:

- **Action Bar Background Opacity:** Use the slider to change the opacity of the action bar. The lower the value, the more see-through it appears.
- **Chat Background Opacity:** Adjust the slider to change the opacity of chat message backgrounds.
- **Text Background Opacity:** Use the slider to choose the opacity of backgrounds for text from the /text command.
- **Chat message duration:** Select from three options to choose how long a message from chat remains visible:
  - 3 seconds
  - 10 seconds (default)
  - 30 seconds
- **Toast notification duration:** Select from three options how long notifications such as game invitations and marketplace offers remain visible:
  - 3 seconds (default)
  - 10 seconds  
  - 30 seconds
- **GUI Scale Modifier:** Select what size the HUD and other UI elements will appear on your screen. Adjust to make these elements easier to use on different devices.
- **Extra large new UI:** Toggle ON to increase the size of the UI for easy reading.
- **Reset to default:** Select Reset to return all accessibility settings to their default values.
