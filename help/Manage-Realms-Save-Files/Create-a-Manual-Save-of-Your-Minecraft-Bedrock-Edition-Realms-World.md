Create manual saves of your Realm world in Minecraft: Bedrock Edition to store a backup copy of your Realm and keep your progress safe.  Manual saves remain in your Realm storage until you delete them, giving you full control over what to keep. 

## Create a Manual Save

1.  Launch Minecraft.
2.  Select **Play.**
3.  Select the **Realms **tab.
4.  Select your Realm and select **Settings**. 
5.  Select **Manage Realm**.
6.  Select **Saves**.
7.  Select **Save active world**. In the menu below, you will see options to select **Manual **or **Automatic**. In **Manual**, the saves you create will appear with the most recent saves at the top. 

## Delete a Manual Save 

1.  Launch Minecraft.
2.  Select **Play.**
3.  Select the **Realms **tab
4.  Select your Realm and select **Settings**. 
5.  Select **Manage Realm**.
6.  Select **Saves**.
7.  Select the **kebob menu button **(three dots) to the right of the save file you wish to delete.
8.  Select **Delete Save**.
