If you have trouble with friend invites in Minecraft: Java Edition, it’s usually because of settings on your Friends List or in your [Xbox privacy options](https://www.xbox.com/en-US/user/settings/privacy-and-safety). Here’s what to do if you can’t send a friend invite, don’t see your friend’s invite, or can’t accept your friend’s invite.

## Why can’t I send a friend invite?

- **Turn on the Friends List**: Launch Minecraft: Java Edition and select the Friends icon from the title screen or pause menu and select **Turn on**. If you initially declined the prompt, go to **Options** \> **Online** \> **Friends List** and set Friends List to **ON**.
- **Check the profile name**: Double-check that you correctly entered your friend’s Java profile name. If they don’t know their profile name, have them follow steps to [View or Change Your In-Game Profile Name in Minecraft](../Manage-Your-Minecraft-Profile/View-or-Change-Your-In-Game-Profile-Name-in-Minecraft.md).
- **Review your Xbox safety settings**: Sign in to [Xbox Official Site: Consoles, Games and Community](https://www.xbox.com/en-US/user/settings/privacy-and-safety) and check the **Xbox and Windows 10 devices Online Safety** \> **You can add friends and follow others** header. Make sure it isn’t set to **Block**.

## Why don’t I see my friend’s invite?

- **Turn on the Friends List**: Go to **Options** \> **Online** \> **Friends List** and make sure Friends List is set to **ON**. It must be on to receive friend requests.
- **Your in-game notifications are off**: Set this **ON** to receive toast notifications when you receive a request. If this is off, you will need to check the requests tab manually.
- **Check the requests tab**: Open the Friends List and check the **requests** tab for pending invites. You may also need to wait up to five minutes or restart your game.
- **Allow incoming requests**: Make sure **Allow Requests** is set to **ON**. If it’s off, players can’t send you friend requests.
- **Confirm the profile name**: Confirm your friend used your correct Java profile name when they sent the request. If they’re unsure, send them your Java profile name as shown in your Friends List.

## Why can’t I accept my friend’s invite?

- **Your friend cancelled the request before you accepted it**: If this happens, you will see the incoming request turn into an outgoing friend request. Your friend will then need to accept your invitation.
- **Your friend has the “Allow Requests” setting turned off**: If a player has the friend list turned on, but the **Allow Requests** setting turned off, other players can't accept their outgoing friend requests.
