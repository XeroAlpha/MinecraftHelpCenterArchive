Today we are shipping the first release candidate for 26.2 and the Chaos Cubed game drop! If no critical issues are found, this will be the version that we ship for the eventual full release.

Happy mining!

## Fixed bugs in 26.2 Release Candidate 1

- [MC-308716](https://bugs.mojang.com/browse/MC-308716) - Pressing the "Friends Screen" key bind while in a world doesn't bring up the friends list

## Get the Release Candidate

Release Candidates are available for Minecraft: Java Edition. To install the Release Candidate, open up the [Minecraft Launcher](https://www.minecraft.net/content/minecraft-net/language-masters/download) and enable snapshots in the "Installations" tab.

**Testing versions can corrupt your world, so please backup and/or run them in a different folder from your main worlds.**
