**Posted:** 25 June 2026

A new hotfix is rolling out to address issues found in the latest update. It will become available on each platform as it’s approved, so please allow a little time for it to reach you.

# Fixes:

## Audio

- Fixed custom streamed audio not playing when its resource pack also contains subpacks ([MCPE-239719](https://bugs.mojang.com/browse/MCPE-239719))

## Graphical

- Fixed custom blocks rendering a stray solid face that could bleed through textures when a model face was hidden by setting one of its UV size dimensions to zero

## Performance and Stability

- Fixed several crashes that could occur during gameplay
- Fixed an issue that could cause the framerate to drop or become locked at 30 FPS while playing ([MCPE-239693](https://bugs.mojang.com/browse/MCPE-239693))
- Fixed an issue where a Book and Quill from pre-26.30 worlds freezes the game ([MCPE-239753](https://bugs.mojang.com/browse/MCPE-239753))

## User Interface

- Players with multiplayer restricted skins can now enter worlds with multiplayer disabled
