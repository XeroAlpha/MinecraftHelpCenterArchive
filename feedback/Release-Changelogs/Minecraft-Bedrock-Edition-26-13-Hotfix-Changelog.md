A new hotfix is rolling out to address issues discovered in the latest update. This update will be rolling out to various platforms as it becomes available, so please allow time for the update to reach you.

## Fixes:

- Fixed several bugs that could affect gameplay
- Fixed an issue with custom entity components locked behind runtime IDs ([MCPE-237713](https://bugs.mojang.com/browse/MCPE/issues/MCPE-237713))
- Selected emotes no longer revert to defaults when relaunching the title ([MCPE-188449](https://bugs.mojang.com/browse/MCPE/issues/MCPE-188449))
