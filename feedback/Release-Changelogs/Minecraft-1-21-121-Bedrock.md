**Posted:**Â 4 November 2025

A new update has been released to address issues discovered in the 1.21.120 release. This update will be rolling out to the various platforms as they become available, so please allow time for the update to reach you.

## **Fixes:**[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki/30606/R21U11-Hotfix-3?anchor=fixes%3A)

## Input[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki?wikiVersion=GBwikiMaster&pagePath=/Bedrock%20Release%20Schedule/Changelogs/Full%20Release%20Changelogs/R21U12%20Hotfix%201&pageId=30831&_a=edit&anchor=input)

- Fixed a bug on Windows where resuming the game with the Task Manager opened caused the cursor to stutter at the center of the screen ([MCPE-229549](https://bugs.mojang.com/browse/MCPE/issues/MCPE-229549))
- Fixed a bug on Windows where the camera snapped to a random direction after closing the GUI and clicking any mouse button ([MCPE-229848](https://bugs.mojang.com/browse/MCPE/issues/MCPE-229848))
- Restored support for input from IME on Windows ([MCPE-229246](https://bugs.mojang.com/browse/MCPE/issues/MCPE-229246))

## Stability and Performance[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki?wikiVersion=GBwikiMaster&pagePath=/Bedrock%20Release%20Schedule/Changelogs/Full%20Release%20Changelogs/R21U12%20Hotfix%201&pageId=30831&_a=edit&anchor=stability-and-performance)

- Fixed a crash from using a data-driven renderer
- Fixed a crash on PlayStation 5 caused by excessive memory use when stacking large add-ons

## User Interface[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki?wikiVersion=GBwikiMaster&pagePath=/Bedrock%20Release%20Schedule/Changelogs/Full%20Release%20Changelogs/R21U12%20Hotfix%201&pageId=30831&_a=edit&anchor=user-interface)

- Fixed a bug where resource pack settings would not apply in-game when changed ([MCPE-229024](https://bugs.mojang.com/browse/MCPE/issues/MCPE-229024))
