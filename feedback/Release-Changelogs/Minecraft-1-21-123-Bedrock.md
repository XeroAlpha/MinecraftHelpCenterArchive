**Posted:** 13 November 2025

A new update has been released to address issues discovered in the 1.21.120 release. This update will be rolling out to the various platforms as they become available, so please allow time for the update to reach you.

> **Developer's Note:** *Huge thanks to everyone who has been sending us bug reports and feedback around the game on Windows! We're actively investigating all the reports coming in and working to improve game quality. We'll keep you all updated on further improvements through these changelogs. Please keep the reports coming in through *[*bugs.mojang.com*](http://bugs.mojang.com/).

 

## Known Issue[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki/30883/R21U12-Hotfix-3?anchor=known-issue)

- Mouse cursor moves in the background during gameplay ([MCPE-230832](https://report.bugs.mojang.com/servicedesk/customer/portal/6/MCPE-230832))
  - Thank you for your patience while we continue to work on this issue. We’re pleased to hear it’s resolved for many of you, but we know more work is needed. Please be assured that we’re actively addressing the remaining problems and aim to have the bug fully fixed as soon as possible.

# Fixes:[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki/30883/R21U12-Hotfix-3?anchor=fixes%3A)

## Performance and Stability[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki/30883/R21U12-Hotfix-3?anchor=performance-and-stability)

- Reduced increased memory utilization from using Improved Input Latency ([MCPE-179496](https://report.bugs.mojang.com/servicedesk/customer/portal/6/MCPE-179496))
- Fixed several crashes that could occur during gameplay 
- Fixed a crash when using hotbar hotkeys while hovering over the Stonecutter preview interface ([MCPE-230558](https://report.bugs.mojang.com/servicedesk/customer/portal/6/MCPE-230558))

## Graphical[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki/30883/R21U12-Hotfix-3?anchor=graphical)

- Fixed an issue where pink textures could occur on Windows devices ([MCPE-167489](https://report.bugs.mojang.com/servicedesk/customer/portal/6/MCPE-167489))

## Input[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki/30883/R21U12-Hotfix-3?anchor=input)

- Fixed an issue where IME Candidates would appear in the wrong monitor or misplaced in relation to text field 
- Fixed IME input when application lost and regained focus (switching from the app and back while editing) 

## World Generation[](https://dev.azure.com/dev-mc/Minecraft/_wiki/wikis/Minecraft.wiki/30883/R21U12-Hotfix-3?anchor=world-generation)

- Blocks with the tick component tick when placed by Jigsaw Structure during world generation
