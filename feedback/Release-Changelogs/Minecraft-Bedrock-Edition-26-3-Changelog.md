**Posted:** 2 March 2026

A new hotfix has been released to address issues discovered in the latest update. This update will be rolling out to the various platforms as they become available, so please allow time for the update to reach you.

## Fixes:

- Fixed an issue with the localization of some items from the Mounts of Mayhem drop. These will now show in the chosen language correctly ([MCPE-235956](https://bugs.mojang.com/browse/MCPE-235956 "https://bugs.mojang.com/browse/mcpe-235956"))
- Fixed a crash that could occur when loading a world or Realm
