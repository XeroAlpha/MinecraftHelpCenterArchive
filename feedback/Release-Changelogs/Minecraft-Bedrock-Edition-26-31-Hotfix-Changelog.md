**Posted:** 22 June 2026

A new hotfix is rolling out to address issues found in the latest update. It will become available on each platform as it’s approved, so please allow a little time for it to reach you.

# Fixes:

## User Interface

- Fixed a bug where players could not join Experiences when the 'Screen Animations' setting was turned off, causing the game to hang on the loading screen

## Gameplay

- Fixed an ID collision when adding split-screen guests on Nintendo Switch, allowing multiple guests to join

## Blocks

- Fixed a bug where projectiles fired from an upward-facing dispenser would not launch into the air ([MCPE-239709](https://bugs.mojang.com/browse/MCPE-239709 "https://bugs.mojang.com/browse/mcpe-239709"))

## Networking

- Fixed an issue causing slower server connectivity issues ([MCPE-239705](https://bugs.mojang.com/browse/MCPE-239705 "https://bugs.mojang.com/browse/mcpe-239705"))

## Performance and Stability

- Fixed several crashes that could occur during gameplay
- Fixed a crash that could occur in the main menu on Switch ([MCPE-238647](https://bugs.mojang.com/browse/MCPE-238647%22https://bugs.mojang.com/browse/mcpe-238647%22))
