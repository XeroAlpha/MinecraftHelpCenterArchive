It's a sunny Wednesday in the shipping room and we're starting the day off with releasing 26.1.1, a hotfix release that addresses an important issue reported in the 26.1 release.

Happy mining!

## Fixed bugs in 26.1.1

- [MC-307140](https://bugs.mojang.com/browse/MC-307140) - Chat messages can no longer be reported if the chat is enabled

## Get the Release

To install the Release, open up the [Minecraft Launcher](https://www.minecraft.net/content/minecraft-net/language-masters/download) and click play! Make sure your Launcher is set to the "Latest Release” option.
