**Posted:** 18 November 2025

### **Information on Minecraft Preview and Beta:**

- These work-in-progress versions can be unstable and may not be representative of final version quality
- Minecraft Preview is available on Xbox, PlayStation, Windows, and iOS devices. More information can be found at [aka.ms/PreviewFAQ](https://aka.ms/PreviewFAQ)
- The beta is available on Android (Google Play). To join or leave the beta, see [aka.ms/JoinMCBeta](https://aka.ms/JoinMCBeta) for detailed instructions

It's time for another Preview and Beta release! For the full list of goodies, browse the changelog below. And as always, we’re keen to get your feedback on these new features at [feedback.minecraft.net](http://feedback.minecraft.net/), and you can report any bugs you find at [bugs.mojang.com](http://bugs.mojang.com/).

# Features and Bug Fixes[](https://feedback.minecraft.net/hc/en-us/articles/anchor=features-and-bug-fixes)

## Mounts of Mayhem[](https://feedback.minecraft.net/hc/en-us/articles/anchor=mounts-of-mayhem)

### Mount Improvements[](https://feedback.minecraft.net/hc/en-us/articles/anchor=mount-improvements)

- Netherite Horse Armor now provides 0.1 points of knockback resistance ([MCPE-230871](https://bugs.mojang.com/browse/MCPE-230871))

### Nautilus Armor[](https://feedback.minecraft.net/hc/en-us/articles/anchor=nautilus-armor)

- Netherite Nautilus Armor now provides 0.1 points of knockback resistance ([MCPE-230871](https://bugs.mojang.com/browse/MCPE-230871))

### Zombie Horse[](https://feedback.minecraft.net/hc/en-us/articles/anchor=zombie-horse)

- The Zombie Horse no longer spawns on Peaceful mode 

## Profile[](https://feedback.minecraft.net/hc/en-us/articles/anchor=profile)

- Removed unexpected Sign Out button on the user profile screen on Windows. Players should sign out via the Xbox app ([MCPE-228176](https://bugs.mojang.com/browse/MCPE-228176))

## Text Input[](https://feedback.minecraft.net/hc/en-us/articles/anchor=text-input)

- Fixed an issue where IME Candidates would appear in the wrong monitor or misplaced in relation to text field

# Technical Updates[](https://feedback.minecraft.net/hc/en-us/articles/anchor=technical-updates)

## Editor[](https://feedback.minecraft.net/hc/en-us/articles/anchor=editor)

- Fixed an issue when trying to move blocks outside of the bounds of the world, the Editor would freeze

## Graphical[](https://feedback.minecraft.net/hc/en-us/articles/anchor=graphical)

- Updated supported devices for Vibrant Visuals on iOS and Android including additional support for the Adreno 700 series ([MCPE-229154](https://bugs.mojang.com/browse/MCPE-229154))
- Fixed an issue where Improved Input Latency Response toggle would cause a crash at startup ([MCPE-231485](https://bugs.mojang.com/browse/MCPE-231485))

## World Generation[](https://feedback.minecraft.net/hc/en-us/articles/anchor=world-generation)

- Blocks with the tick component tick when placed by Jigsaw Structure during world generation\
